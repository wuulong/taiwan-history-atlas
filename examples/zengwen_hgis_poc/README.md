# 曾文溪 HGIS 考古熱點分析 POC 成果釋出

本範例展示了如何運用 QGIS 對曾文溪流域的考古遺址進行進階空間分析，包含密度熱點與高程分佈研究。

## 目錄說明
- `Zengwen_HGIS_Hub.qgs`: QGIS 專案檔，已配置好所有樣式與圖層。
- `poc_data_hub.db`: 本次分析專用的 SQLite 數據快照，包含經過處理的考古遺址數據。
- `poc_data_hub.vrt`: OGR 虛擬格式定義檔，負責將快照資料庫介接至 QGIS。
- `hgis_sites.vrt`: OGR 虛擬格式定義檔，用於連結儲存庫共用的 `taiwan_history.db` 聚落資料。
- `0_曾文溪全流域整合圖資_WGS84.kml`: 曾文溪流域範圍、水系與河道圖資。

## 使用說明
1. 請使用 **QGIS 3.x** 版本開啟 `Zengwen_HGIS_Hub.qgs`。
2. 開啟前，請確保您的目錄結構中包含 `../../data/taiwan_history.db`（若您完整克隆了 `taiwan-history-atlas` 儲存庫，該檔案應已存在）。
3. 本專案採用「相對路徑」配置，直接解壓縮或克隆後即可一鍵開啟分析畫面。

## 分析特點
- **遺址密度熱點 (Heatmap)**：根據遺址重要度 (importance_rank) 加權計算出的密度雲圖。
- **高程分級圖層 (Graduated)**：將遺址依海拔高度分組，展現人類聚落與地貌的互動規律。
- **歷史底圖套疊**：預設介接「中研院台灣百年歷史地圖」之日治時期堡圖 (WMTS)。
